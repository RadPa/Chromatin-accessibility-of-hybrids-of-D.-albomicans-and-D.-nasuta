# HMMRATAC v1.2.10

ATAC-seq specific peak calling using HMMRATAC (Tarbell and Liu, 2019).

## Requirements
1. `shifted.bam` from **ATACseqQC**
2. **Mitochondrial reads** were removed from `shifted.bam`
3. Genome size
4. Blacklist

### Contents
1. Size of genome
2. Log files of peak caller
3. Peak files
4. Script
