# Pre-alignment QC

## FastQC
The raw reads quality were analyzed using **FastQC v0.11.9**

## Adaptor removal
The adaptor contamination of the reads were removed using **cutadapt v2.8**

## MultiQC
Log files of FastQC and cutadapt were analyzed using **MultiQC v1.9**
